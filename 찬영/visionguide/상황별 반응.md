# 상황별 반응

## 음성 우선순위

> 여러 상황이 동시에 발생하면 아래 순서대로 높은 우선순위 음성이 먼저 발화되고, 낮은 순위는 대기하거나 생략됨

| 순위 | 상황 | 예시 |
|---|---|---|
| 1 | 장애물 긴급 (1m 이하) | "위험! 앞 차량 매우 가깝습니다." |
| 2 | 횡단보도 진입 | "횡단보도입니다. 멈추고 신호를 확인하세요." |
| 3 | 장애물 5m 진입 음성 | "왼쪽 2.3m 차량 접근" |
| 4 | 차도 / 자전거도로 진입 | "차도입니다. 주의하세요." |
| 5 | 골목 / 주의구역 진입 | "골목길입니다. 주의하세요." |
| 6 | 인도 진입 | "인도입니다." |
| 7 | 차도 / 자전거도로 5초 반복 유도 | "차도입니다. 왼쪽 인도로 이동하세요." |

## 장애물 (물체)

| 그룹 | 하위 항목 | 5m 이내 진입 | 3m 이내 | 1m 이하 |
|---|---|---|---|---|
| **차량** | 승용차, 버스, 트럭, 오토바이 | 음성 1회 + 진동(약) | 진동(중) | 긴급 음성 1회 + 진동(최대) |
| **개인이동장치** | 자전거, 스쿠터 | 음성 1회 + 진동(약) | 진동(중) | 긴급 음성 1회 + 진동(최대) |
| **사람/동물** | 사람, 고양이, 개, 유모차, 휠체어, 리어카, 이동식 안내판 | 없음 | 없음 | 긴급 음성 1회 |
| **고정장애물** | 바리케이드, 벤치, 볼라드, 의자, 소화전, 키오스크, 주차미터, 기둥, 화분, 전력 제어기, 정류장, 테이블, 가로수, 신호등 제어기 | 없음 | 없음 | 긴급 음성 1회 |
| **신호등/표지판** | 신호등, 교통표지판 | 없음 | 없음 | 진동(최대) |
| 기타 | — | 없음 | 없음 | 없음 |

## 노면

> 노면 종류가 바뀔 때마다 1회 음성, 같은 노면을 유지하는 동안엔 반복 없음
> 단, ⚠️ 표시 항목은 머무는 동안 5초마다 반복

| 그룹 | 하위 항목 | 진입 시 음성 | 계속 있을 때 |
|---|---|---|---|
| **횡단보도** | 차도 횡단보도 / 골목 횡단보도 | "횡단보도입니다. 멈추고 신호를 확인하세요." | 신호등 상태 음성 (아래 참고) |
| **차도** ⚠️ | 차도 | "차도입니다. 주의하세요." | 5초마다 방향 유도 음성 (아래 참고) |
| **골목** | 골목길 / 골목(파손) / 과속방지턱 | "골목길입니다. 주의하세요." | 없음 |
| **주의구역** | 파손 인도 | "파손된 인도입니다. 주의하세요." | 없음 |
| **주의구역** | 점자블록(파손) | "파손된 점자블록입니다. 주의하세요." | 없음 |
| **주의구역** | 격자 덮개 | "격자 덮개입니다. 주의하세요." | 없음 |
| **주의구역** | 맨홀 | "맨홀이 있습니다. 주의하세요." | 없음 |
| **주의구역** | 공사 구역 | "공사 구역입니다. 주의하세요." | 없음 |
| **주의구역** | 계단 | "계단입니다. 주의하세요." | 없음 |
| **주의구역** | 나무 구역 | "나무 구역입니다. 주의하세요." | 없음 |
| **점자블록** | 점자블록(정상) | 없음 | 없음 |
| **인도** | 인도(아스팔트/블록/시멘트/우레탄/흙돌/기타) | "인도입니다." | 없음 |
| **자전거도로** ⚠️ | 자전거 도로 | "자전거 도로입니다. 주의하세요." | 5초마다 방향 유도 음성 (아래 참고) |

### 방향 유도 음성 (차도 · 자전거도로 5초 반복)

> 카메라로 감지된 왼쪽/오른쪽 노면 정보를 기반으로 동적 생성

| 왼쪽 | 오른쪽 | 유도 음성 |
|---|---|---|
| 인도 or 골목 | 없음 | "{상황} 왼쪽 인도(골목)로 이동하세요." |
| 없음 | 인도 or 골목 | "{상황} 오른쪽 인도(골목)로 이동하세요." |
| 인도 or 골목 | 인도 or 골목 | "{상황} 왼쪽 또는 오른쪽 인도로 이동하세요." |
| 횡단보도 | 없음 or 횡단보도 | "{상황} 왼쪽(오른쪽) 횡단보도 방향으로 이동하세요." |
| 주의구역 | 없음 or 주의구역 | "{상황} 왼쪽(오른쪽) 골목 방향으로 이동하세요." |
| 없음 | 없음 | "{상황} 천천히 멈추고 주변을 확인하세요." |

- {상황} = "차도입니다." 또는 "자전거 도로입니다."

### 횡단보도 신호등 상태

> 횡단보도 진입 후 카메라에 신호등이 보이면 색상을 분석해 안내

| 신호등 상태 | 음성 |
|---|---|
| 초록불 | "초록불입니다. 건너세요." |
| 빨간불 | "빨간불입니다. 멈추세요." |
| 신호등 없음 / 인식 불가 | "신호등을 확인할 수 없습니다. 주변을 살펴보세요." → 왼쪽·오른쪽에 차량·개인이동장치 없으면 "건너셔도 됩니다." |

## 목적지 네비게이션

> 목적지 설정 후 GPS 기반으로 경로 안내, 이탈 시 재탐색

| 상황 | 음성 |
|---|---|
| 올바른 방향으로 이동 중 | "X미터 앞에서 좌회전(우회전)하세요." |
| 반대 방향으로 걷는 중 | "반대 방향입니다. 돌아서세요." |
| 경로 크게 이탈 | "경로를 벗어났습니다. 경로를 재탐색합니다." |
| 경로 재탐색 완료 | "새로운 경로를 안내합니다." |
| 교차로 접근 | "X미터 앞에서 좌회전(우회전)하세요." |
| 목적지 근처 | "목적지가 X미터 앞입니다." |
| 목적지 도착 | "목적지에 도착했습니다." |

## 서버 연결 상태

> 서버와의 연결 여부에 따라 사용 가능한 기능이 달라짐

| 상황 | 음성 | 사용 가능 기능 |
|---|---|---|
| **앱 시작 - 연결 정상** | "서버에 연결되었습니다. 모든 기능을 사용할 수 있습니다." | 전체 기능 |
| **앱 시작 - 연결 안 됨** | "서버에 연결할 수 없습니다. 장애물 탐지, 노면 안내, 신호등 인식 기능을 사용할 수 없습니다." | GPS 네비게이션, TTS만 동작 |
| 안내 중 연결 끊김 | "서버 연결이 끊겼습니다. 장애물 탐지, 노면 안내, 신호등 인식 기능을 사용할 수 없습니다. 주의하세요." | GPS 네비게이션, TTS만 동작 |
| 재연결 성공 | "서버에 다시 연결되었습니다. 모든 기능을 사용할 수 있습니다." | 전체 기능 복구 |
| 재연결 실패 (계속 끊김) | 30초마다 "서버에 연결할 수 없습니다. 주의하세요." | GPS 네비게이션, TTS만 동작 |

### GPS 사용 불가

| 상황 | 음성 | 동작 변화 |
|---|---|---|
| 앱 시작 - GPS 꺼져 있음 | "위치 정보를 사용할 수 없습니다. 목적지 안내 기능을 사용할 수 없습니다." | 장애물 탐지, 노면 안내는 정상 동작 |
| 안내 중 GPS 끊김 | "위치 정보 연결이 끊겼습니다. 목적지 안내를 사용할 수 없습니다. 주의하세요." | 목적지 안내 중단 |
| GPS 다시 켜짐 | "위치 정보가 연결되었습니다. 목적지 안내를 사용할 수 있습니다." | 정상 안내로 복구 |

### 거리 측정 (Depth) 사용 불가

| 상황 | 음성 | 동작 변화 |
|---|---|---|
| Depth 꺼짐 / 실패 (앱 시작 시) | "거리 측정을 사용할 수 없습니다. 장애물 거리를 알 수 없습니다." | 거리 없이 방향만 안내 |
| Depth 꺼진 상태에서 장애물 탐지 | "왼쪽(정면/오른쪽)에 {장애물}이 있습니다." | 5m·1m 단계 구분 없이 탐지 즉시 1회 음성 |
| Depth 다시 켜짐 | "거리 측정이 활성화되었습니다." | 정상 거리 기반 안내로 복구 |

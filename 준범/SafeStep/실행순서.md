# SafeStep 실행 순서 (팀원용)

---

## 처음 세팅하는 경우

### 1단계 — 레포 클론

```powershell
git clone https://github.com/milk152milk/navigation.git
```

> 클론하면 모델 파일(bbox.pt, surface.pt)을 포함한 모든 파일이 자동으로 다운로드됩니다. 별도로 파일을 받을 필요 없습니다.

---

### 2단계 — 서버 패키지 설치 (최초 1회)

Python 3.9 이상이 설치되어 있어야 합니다.

```powershell
pip install fastapi uvicorn ultralytics opencv-python pillow numpy
```

---

### 3단계 — PC Wi-Fi 연결

> **핵심: PC와 폰이 반드시 같은 공유기(Wi-Fi)에 연결되어 있어야 합니다.**

윈도우 우측 하단 네트워크 아이콘 → Wi-Fi 켜기 → 폰과 같은 공유기 선택 후 연결

---

### 4단계 — PC의 Wi-Fi IP 확인

```powershell
ipconfig
```

출력 결과에서 **Wireless LAN adapter Wi-Fi** 항목의 IPv4 주소를 메모합니다.

```
Wireless LAN adapter Wi-Fi:
   IPv4 Address. . . : 192.168.0.XX   ← 이 주소
```

---

### 5단계 — 앱에 IP 입력

Android Studio에서 아래 파일을 엽니다:

```
navigation/준범/SafeStep/app/src/main/java/com/safestep/app/detect/RemoteDetector.kt
```

SERVER_URL의 IP를 4단계에서 확인한 IP로 수정합니다:

```kotlin
const val SERVER_URL = "http://192.168.0.XX:8000/detect"
//                              ↑ 본인 Wi-Fi IP로 변경
```

---

### 6단계 — 앱 빌드

Android Studio에서 프로젝트를 엽니다:

```
navigation/준범/SafeStep
```

폰을 USB로 연결한 뒤 상단 ▶ Run 버튼 클릭 (또는 Shift+F10)

앱 최초 실행 시 아래 권한을 모두 **허용** 해주세요:
- 카메라
- 마이크
- 위치

---

### 7단계 — 서버 실행

```powershell
cd navigation/준범/SafeStep/server
python server.py
```

아래와 같이 출력되면 정상입니다:

```
[SafeStep] YOLO 탐지 로드 완료 ✅
[SafeStep] 노면 세그멘테이션 로드 완료 ✅
[SafeStep] 서버 시작 (포트 8000)
INFO: Uvicorn running on http://0.0.0.0:8000
```

---

### 8단계 — 연결 확인

폰 브라우저에서 아래 주소 접속:

```
http://[PC Wi-Fi IP]:8000/health
```

아래와 같이 뜨면 정상 연결된 것입니다:

```json
{"status": "ok", "yolo": "bbox.pt", "surface": "surface.pt"}
```

---

## 매일 실행할 때

```
① PC Wi-Fi 연결 확인 (폰과 같은 공유기)
② ipconfig 로 Wi-Fi IP 확인
③ IP 바뀌었으면 RemoteDetector.kt 수정 후 앱 다시 빌드
④ python server.py 로 서버 실행
⑤ 앱 실행
```

> Wi-Fi IP는 재접속할 때마다 바뀔 수 있습니다. 매번 확인하는 습관이 필요합니다.

---

## 서버 강제 종료

```powershell
Get-Process python | Stop-Process -Force
```

---

## 방화벽 설정 (연결 안 될 때 1회 실행, 관리자 PowerShell)

```powershell
New-NetFirewallRule -DisplayName "SafeStep Server" -Direction Inbound -Protocol TCP -LocalPort 8000 -Action Allow
```
